cURL Command Examples

1. Register a User

curl -X POST http://YOUR_VM_IP:3000/register \ -H "Content-Type: application/json" \ -d '{"username": "newuser", "password": "newpassword123"}'

2. Login (Get JWT Token)

curl -X POST http://YOUR_VM_IP:3000/login \ -H "Content-Type: application/json" \ -d '{"username": "testuser", "password": "test123"}'

3. Access Protected Route

curl -X GET http://YOUR_VM_IP:3000/protected \ -H "Authorization: Bearer YOUR_JWT_TOKEN"

4. Access Admin Route

curl -X GET http://YOUR_VM_IP:3000/admin \ -H "Authorization: Bearer YOUR_JWT_TOKEN"

5. Get User Profile

curl -X GET http://YOUR_VM_IP:3000/profile \ -H "Authorization: Bearer YOUR_JWT_TOKEN"

6. Refresh Token

curl -X POST http://YOUR_VM_IP:3000/refresh \ -H "Authorization: Bearer YOUR_JWT_TOKEN"
